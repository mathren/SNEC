Copy of the SNEC code for further development. Original distribution
source is at [[https://sntheory.org/SNEC.html]].

The idea is to track an inner "ghost cell" which always has velocity
and pressure of the cell above (first in the computational domain), so
that no velocity or pressure gradient is created and no back reaction
applies in the domain.
